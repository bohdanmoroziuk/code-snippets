# Task scripts

## Start the app in development mode

```bash
npm run serve
```

## Build the application for production

```bash
npm run build
```

## Run unit tests

```bash
npm run test:unit
```

## Run e2e tests

```bash
npm run test:e2e
```

## Run eslint and report any linting errors

```bash
npm run lint:report
```

## Run eslint and fix any linting errors

```bash
npm run lint:fix
```
